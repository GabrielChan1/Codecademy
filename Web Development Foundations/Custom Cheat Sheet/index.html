Gabriel's HTML Cheat Sheet

Tag Name Uses Format Notable Attributes Prerequisites
HTML Acts as the root of the HTML document. Should contain all HTML code. 
<html></html>
 None None
Head Creates a header section for the website. Generally contains metadata(data about the website) and links to external resources like 
.css
files. 
<head></head>
 None None
Title Defines the title of the HTML document. 
<title></title>
 None Needs to be a child element of a head tag
Body Creates a body section for the website which contains the elements of the website that are visible to the user. 
<body></body>
 None None
Footer Creates a footer section for the website. Generally contains contact information, copyright information, links to other pages, etc.. 
<footer></footer>
 None None
Figure Creates a section for encapsulating media such as images or videos along with any captions or descriptions of them. 
<figure></figure>
 None None
Figure Caption A container for descriptions of media such as images or videos. 
<figcaption></figcaption>
 None Generally created as a child element of a figure tag
Section Defines a section in a website. Typically used to define elements such as chapters, headings, etc. that share the same theme. 
<section></section>
 None None
Article A container for independent, self contained content that makes sense on its own. Typically used for forum posts, blog posts and news stories. 
<article></article>
 None None
Aside A container for additional, optional information that could enhance another element. 
<aside></aside>
 None None
Anchor An inline container for attaching hyperlinks to content. 
<a></a>
 
  • href
    : Contains the link to be attached to the contained content.
 None
Division Divides an HTML document into sections and contains the content within these sections. 
<div></div>
 None None
Span An inline container for grouping text for styling purposes. 
<span></span>
 None None
Paragraph Contains and displays a block of text. 
<p></p>
 None None
Line Break Creates a line break in a body of text. 
<br>
 None None
Heading Increases the size of the text contained inside depending on the level used. 
<h1></h1>

<h2></h2>

<h3></h3>

<h4></h4>

<h5></h5>

<h6></h6>
 None None
Ordered List Creates an ordered list section where list elements could be added. 
<ol></ol>
 None None
Unordered List Creates an unordered list section where list elements could be added. 
<ul></ul>
 None None
List Item Adds a list element to a list section. 
<li></li>
 None Needs to be a child element of a list tag
Emphasis Italicizes the text contained inside. 
<em></em>
 None None
Strong Bolds the text contained inside. 
<strong></strong>
 None None
Image Displays an image on the website. 
<img>
 
  • src
    : Contains the address or path to the image being displayed.
  • alt
    : Alternate content to be displayed if the image cannot be displayed.
 None
Video Displays a video on the website. 
<video></video>
 
  • src
    : Contains the address or path to the video being displayed.
  • width
    : Sets the width of the video player.
  • height
    : Sets the height of the video player.
  • controls
    : Sets the controls for the video player. Setting no value will give basic controls.
 None
Link Import resources such as 
.css
files, fonts or backgrounds to be used in the html code.		 
<link>
 
  • href
    : Contains the address or path to the resource being linked.
  • rel
    : Describes the relationship between the html and css file.
 None
Table Creates a table. 
<table></table>
 
  • border
    : Wraps the table in a border.
 None
Table Header Creates a header section for a table. 
<thead></thead>
 None Needs to be a child element of a table tag
Table Body Creates a body section for a table. 
<tbody></tbody>
 None Needs to be a child element of a table tag
Table Footer Creates a footer section for a table. 
<tfoot></tfoot>
 None Needs to be a child element of a table tag
Table Row Creates a table row. 
<tr></tr>
 None Needs to be a child element of a table tag
Table Header Cell Creates a cell in a table to be used as a header. 
<th></th>
 
  • scope
    : Classifies the purpose of a row or column. Also used by screen readers to improve accessibility for visually impaired users.
 Needs to be a child element of a table tag
Table Data Cell Creates a cell in a table to hold data. 
<td></td>
 
  • colspan
    : Sets the number of columns this data cell will occupy.
  • rowspan
    : Sets the number of rows this data cell will occupy.
  • border
    : Wraps the cell in a border.
 Needs to be a child element of a table tag
